# Teoría básica - Lab 03 .NET 10 API

## ¿Qué es?
Una Web API en .NET expone endpoints HTTP para operaciones CRUD, autenticación y reglas de negocio.

## Palabras clave
- ASP.NET Core
- Controller
- DTO
- Dependency Injection
- EF Core
- JWT

## Comandos clave
```bash
dotnet restore
dotnet run
dotnet test
```

## Buenas prácticas
- Separar capas (API, aplicación, infraestructura).
- Usar DTOs para entrada/salida.
- Validar y manejar errores de forma uniforme.
