# Teoría básica - Lab 06 Angular 21

## ¿Qué es?
Angular es un framework frontend para construir aplicaciones SPA robustas. En Angular 21, Signals permite manejo de estado reactivo más simple.

## Palabras clave
- Angular
- Signals
- Standalone Components
- HttpClient
- Interceptor
- Reactive Forms

## Comandos clave
```bash
npm install
npm start
npm run build
```

## Buenas prácticas
- Usar servicios para acceso a API.
- Centralizar errores HTTP con interceptor.
- Separar estado (store/signals) de componentes.
