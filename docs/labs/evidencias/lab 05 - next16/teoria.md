# Teoría básica - Lab 05 Next.js 16

## ¿Qué es?
Next.js es un framework de React para aplicaciones full-stack con renderizado híbrido y rutas modernas.

## Palabras clave
- App Router
- Server Components
- Client Components
- Server Actions
- SSR

## Comandos clave
```bash
npm install
npm run dev
npm run build
```

## Buenas prácticas
- Diferenciar lógica de servidor y cliente.
- Proteger rutas y sesiones.
- Consumir APIs con manejo de errores y estados de carga.
