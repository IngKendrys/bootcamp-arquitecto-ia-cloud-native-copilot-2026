# Teoría básica - Lab 02 Codespaces

## ¿Qué es?
GitHub Codespaces es un entorno de desarrollo en la nube, listo para usar, basado en contenedores.

## Palabras clave
- Dev Container
- Remote development
- Port forwarding
- Dotfiles
- Prebuilds

## Comandos/acciones clave
- Crear Codespace desde el repositorio.
- Abrir terminal y ejecutar `npm install`, `dotnet restore`, `uvicorn`, etc.
- Publicar puertos para acceder a apps web.

## Buenas prácticas
- Definir `devcontainer.json` con herramientas necesarias.
- Usar puertos y variables de entorno consistentes.
- Mantener dependencias reproducibles.
