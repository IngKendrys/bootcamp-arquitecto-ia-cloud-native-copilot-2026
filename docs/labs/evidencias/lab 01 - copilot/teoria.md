# Teoría básica - Lab 01 Copilot

## ¿Qué es?
GitHub Copilot es un asistente de IA para desarrollo que sugiere código, pruebas, documentación y refactors dentro del editor.

## Palabras clave
- Prompt
- Contexto
- Autocompletado
- Chat
- Refactor
- Test generation

## Comandos/acciones clave
- Abrir chat de Copilot en VS Code.
- Pedir generación de funciones, tests o explicación de código.
- Solicitar refactor de legibilidad y mantenibilidad.

## Buenas prácticas
- Dar contexto claro (archivo, lenguaje, objetivo).
- Pedir cambios pequeños e iterativos.
- Validar siempre con build/test.
