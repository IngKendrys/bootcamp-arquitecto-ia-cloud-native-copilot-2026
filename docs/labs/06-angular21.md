# Lab 06 - Angular 21 frontend alternativo

## Objetivo
Implementar un frontend Angular que consuma los mismos endpoints del backend.

## Prerrequisitos
- API disponible de Lab 03 o 04.
    - **Ubicación**: La API se encuentra en `templates/dotnet-api` (Lab 03) o `templates/node-api` (Lab 04).
    - **Ejecución**: Navega al directorio correspondiente e instala dependencias con `npm install` (Node) o restaura el proyecto (.NET). Luego ejecuta `npm start` (Node) o `dotnet run` (.NET).
    - **Verificación**: Confirma que la API responde en `http://localhost:3000` (Node) o `http://localhost:5000` (.NET).

## Paso a paso

2. Crea componentes de listado con paginación y filtro, y componente de formulario.
3. Implementa validaciones de formulario.
